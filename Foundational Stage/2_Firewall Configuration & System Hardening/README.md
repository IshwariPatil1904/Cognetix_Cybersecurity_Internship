# Firewall Configuration & System Hardening

## Overview
This project focuses on securing a Windows system through firewall configuration and basic hardening practices. It includes evidence of policy implementation and system-level security settings.

## Contents
- Report folder with the written task documentation
- Screenshots showing firewall rules, Defender configuration, and hardening steps

## Main Hardening Activities
- Enabled Windows Defender Firewall
- Created allow rules for required services on ports 22, 80, and 443
- Blocked unnecessary or risky ports such as 21 and 23
- Disabled the guest account
- Verified Windows Update configuration
- Reviewed password policy settings

## Purpose
The goal of this task was to demonstrate practical system hardening by reducing exposure to unauthorized access and enforcing safer default security settings.
