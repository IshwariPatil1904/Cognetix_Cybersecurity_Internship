# ✅ Completed Projects

## 🔍 Project 1 - System Vulnerability Assessment Using Nmap

### Objective
Perform vulnerability assessment and network reconnaissance on a target host using **Nmap** to identify:

- Live hosts
- Open ports
- Running services
- OS-level indicators
- Potential vulnerabilities

### Key Activities

- Host discovery and target identification
- TCP/service scanning
- Aggressive and vulnerability script scans
- Documentation of findings and evidence

### Deliverables

- Scan result files
- Screenshots
- Project report
- Security observations

---

## 🔥 Project 2 - Firewall Configuration & System Hardening

### Objective
Improve Windows endpoint security by applying firewall controls and foundational hardening practices.

### Key Activities

- Enabled and reviewed Windows Defender Firewall settings
- Created rules for required services
- Blocked unnecessary ports
- Performed account and update checks
- Documented hardening evidence

### Deliverables

- Configuration screenshots
- Hardening evidence
- Project report
- Security recommendations

---

## 🔐 Project 3 - Password Security Analysis & Policy Design

### Objective
Use Python to evaluate password strength characteristics and design a practical password policy.

### Key Activities

- Loaded and analyzed password dataset
- Assessed strength patterns and reuse trends
- Demonstrated secure hashing using bcrypt
- Built visualizations in Jupyter Notebook
- Drafted password policy document

### Deliverables

- Dataset and notebook
- Password policy document
- Generated analysis reports
- Screenshots and final report
